# 遺忘之森 (Forest of Oblivion) - 專案說明文件

《遺忘之森》是一款結合了養成、抽卡、回合制戰鬥與放置系小遊戲的 Java Swing 桌面應用程式。玩家將扮演森林的守護者，透過淨化黑化怪獸來收集夥伴，並經營自己的小屋與資產。

---

## 🛠 環境建置需求

* **開發語言**: Java (建議 JDK 8 或更高版本)
* **執行環境**: Java Runtime Environment (JRE)
* **圖形介面庫**: Swing (內建於 JDK)
* **音訊支援**: Java Sound API (需支援 `.wav` 檔案)
* **儲存方式**: 使用 Java 序列化 (`ObjectOutputStream`) 進行 `save.dat` 本地存檔。

## 🎮 核心功能介紹

1. **抽卡系統 (Gacha)**: 消耗「卡幣」抽取夥伴。設有保底機制（每 30 抽必得 SSSR 角色）。
2. **戰鬥淨化**: 透過回合制戰鬥擊敗黑化怪獸，成功後可將怪獸轉化為新夥伴。
3. **夥伴養成**:
* **送禮系統**: 贈送禮物提升好感度，進而增加夥伴的 ATK 與 HP。
* **屬性加成**: 好感度越高，戰鬥時越強大。


4. **商城與小屋**: 可購買家具（森之呼吸、霓虹核心風格）以及各式禮物。
5. **資源採集**: 「森之工作坊」放置小遊戲，可自動化生產卡幣、禮幣與家幣。
6. **圖鑑系統**: 追蹤收集進度，包含夥伴、黑化怪獸及淨化夥伴的詳細資訊。
7. **系統設定**: 提供音量調整 (BGM/SFX)、對話框色彩與字體大小客製化功能。

## 📖 遊玩說明

* **啟動遊戲**: 執行 `final_game.java` 的 `main` 方法。
* **新手起步**: 點擊「開始遊戲」，透過抽卡獲得初始夥伴。
* **挑戰關卡**: 在主介面點擊「地圖」，依序挑戰黑化怪獸。建議確認夥伴實力後再進入。
* **獲取資源**: 前往「工作」進行資源採集，可利用自動模式節省時間。
* **提升戰力**: 在「夥伴」介面中查看擁有夥伴，並透過「送禮物」提升好感，好感度升級時會自動提升夥伴能力。
* **資料保存**: 遊戲具備自動儲存機制，退出時會自動更新 `save.dat`，確保進度不遺失。
